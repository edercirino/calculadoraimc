## Como executá-lo

Se você for programador, faça os seguintes passos: 

Esteja certo que sua máquina possua:
 * git
 * nodejs versão 10>
 * yarn
  
Rodar os comandos:
```
git clone git@github.com:edercirino/calculadoraimc.git
yarn install
npx react-native start
npx react-native run-android
```
